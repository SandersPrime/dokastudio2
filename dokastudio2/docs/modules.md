# Модули платформы

## Основные модули

### Auth (Аутентификация)
- Регистрация и вход пользователей
- JWT-аутентификация
- Роли: ADMIN, AUTHOR, USER
- Восстановление пароля

### Upload (Загрузки)
- Загрузка медиафайлов для вопросов
- Ограничение по типу и размеру
- Хранение в директории `uploads/`
- Защита от неавторизованного доступа

### Quizzes (Викторины)
- Создание, редактирование, удаление квизов
- Управление вопросами
- Настройка параметров квиза
- Публикация и черновики

### Questions (Вопросы)
- Поддержка разных типов вопросов:
  - Текст
  - Изображение
  - Аудио
  - Видео
  - Правда/Ложь
  - Вопрос с выбором
  - Сопоставление
  - Рейтинг
  - Облако слов
- Расширенные настройки (тайминг, подсчёт баллов)

### Sessions / Live (Сессии)
- Проведение живых викторин
- Роли: ведущий и игрок
- Код подключения (PIN)
- Реалтайм-взаимодействие через Socket.IO
- Отображение результатов в реальном времени

### Homework (Домашние задания)
- Создание заданий на основе квизов
- Генерация PIN-кода
- Отслеживание прогресса учеников
- Просмотр результатов

### Catalog (Каталог)
- Просмотр публичных квизов
- Покупка шаблонов
- Использование в конструкторе

### Marketplace (Маркетплейс)
- Публикация квизов для продажи
- Модерация (для администраторов)
- Категории и теги
- Ценообразование

### Earnings (Заработок)
- Отслеживание доходов авторов
- Запросы на вывод средств
- История транзакций
- Админ-панель для модерации выводов

### Equipment (Оборудование)
- Каталог оборудования для стримов
- Заявки на приобретение
- Админ-панель управления товарами
- Уведомления о новых заявках

## Вспомогательные модули

### Flashcards (Флешкарточки)
- Генерация карточек из вопросов квиза
- Режим обучения
- Повторение по интервальному методу

### AI Generator
- Генерация вопросов с помощью ИИ
- Поддержка разных тематик
- Интеграция с внешними API

### Leaderboard (Таблица лидеров)
- Отображение результатов в сессиях
- Рейтинг игроков
- Анимации при обновлении

### Notifications (Уведомления)
- Системные уведомления
- Пуш-уведомления (если реализовано)
- Всплывающие сообщения на фронтенде

## Архитектурные модули

### Socket Architecture
- Разделение событий по ролям (host/player)
- Централизованная регистрация
- Защитные механизмы (guards)
- Обработка отключений

### Constructor Architecture
- Управление состоянием редактора
- Валидация данных
- Сохранение и синхронизация
- Поддержка перетаскивания вопросов

### State Management
- Глобальное состояние приложения
- Синхронизация между компонентами
- Восстановление состояния после перезагрузки

### API Layer
- RESTful API для всех модулей
- Валидация входных данных
- Обработка ошибок
- Rate limiting

### Security
- Аутентификация и авторизация
- Валидация сессий
- Защита от XSS и CSRF
- Валидация файлов и загрузок

### Logging & Monitoring
- Логирование запросов
- Мониторинг производительности
- Отслеживание ошибок
- Health-check эндпоинт

# 🎮 DokaStudio 2.0

Профессиональная платформа для создания и проведения квизов, викторин и тестов.

## 🚀 Возможности

- **Конструктор квизов** — Drag & Drop, типы вопросов (текст, картинки, аудио, видео, правда/ложь)
- **Режим ведущего** — QR-код, PIN, управление игрой в реальном времени
- **Интерфейс игрока** — подключение по коду, таймер, турнирная таблица
- **Флэш-карточки** — создание наборов для запоминания
- **Домашние задания** — асинхронное прохождение с дедлайном
- **ИИ-генерация** — создание квизов через OpenAI
- **Маркетплейс** — продажа и покупка готовых квизов (в разработке)

## 🛠 Технологии

- **Backend:** Node.js + Express
- **Real-time:** Socket.IO
- **База данных:** SQLite + Prisma
- **Авторизация:** JWT
- **Фронтенд:** HTML5, CSS3, JavaScript (модульная архитектура)

## 📦 Установка

```bash
# Клонировать репозиторий
git clone https://github.com/yourusername/dokastudio.git
cd dokastudio

# Установить зависимости
npm install

# Создать .env файл из примера
cp .env.example .env

# Настроить базу данных
npx prisma db push

# Запустить сервер
node app.js
